# Muire Design

React frontend for Muire Design — spatial architecture and interior design. Built with Vite, React, and Tailwind CSS.

## Structure

- `frontend/` — Vite + React app
  - `src/components/Layout/` — Navbar, Footer
  - `src/components/Sections/` — Hero, Philosophy, Spaces, Services
  - `src/pages/` — HomePage
  - `src/styles/globals.css` — Global utility classes
  - `src/config/api.js` — API config

## Run

```bash
cd frontend
npm install
npm run dev
```

Build: `npm run build`
